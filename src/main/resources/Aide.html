Aide

Si vous éprouvez des difficultés à faire fonctionner l'application, essayez les étapes suivantes:

  • Fermez l'application et redémarrez-là;
  • Vérifiez que votre connexion Internet est fonctionnelle;
  • Réessayez plus tard. Les API utilisés sont limitées par un nombre de requêtes maximums par jour. Il se peut que la limite soit atteinte.

Si une erreur perssiste, mentionnez-le sur GitHub.
